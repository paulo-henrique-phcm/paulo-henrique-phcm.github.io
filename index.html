Docker

Alguns exemplos de projetos que ja concretizei usando docker

Profissionalmente já criei e implementei soluções usando API's com Python e Docker. Como são projetos sem inerface e restritas ao cliente, somente descreverei o que foi feito:

• Download automático de XML's de notas fiscais através da API do portal SIEG. A aplicação em container em atividade até hoje no servidor de produção do cliente

• Construí um framework em Python usando Selenium que facilita a criação e manutenção de RPA's Web. O objetivo foi facilitar a automação de processos web em sistemas nos clientes. A solução conta com módulos de conexão e comunicação com bancos de dados, API's, Envio de e-mail, registro de logs, e outros...

• Criei uma extensão para VsCode desenvolvida em Javascript, usada pela empresa e alguns clientes. Essa extensão realiza a análise dos códigos ADVP e TLPP usando uma aplicação JAVA feito pela TOTVS e disponibilizada via container Docker. Para isso, construí uma API com Python/Flask em container, onde adicionei extraí a aplicação java, que recebe via requisição os fontes codificados em base64, executa a aplicação da TOTVS que os analisa e retorna como resposta o resultado da analise em Json. A extensão por sua vez prepara todos os resultados e exibe numa interface interativa web no próprio VsCode. A API está hospedada num servidor GCP e é acessada pela extensão do VsCode, enviando os fontes e retornando o resultado das análises em uma tela personalizada no VsCode, facilitando o desenvolvimento de toda a equipe. Ela é usada até hoje.

Aplicação Desktop

Automação para baixar uma lista de XML's de notas fiscais no portal SIEG

O portal sieg dificulta o processo de download de XML's específicos. Então desenvolvi esta aplicação desktop que acessa o portal com seu usuário e baixa todos os XML's que você forneceu após verificar sua validade, claro!

A aplicação desktop foi feita em Python usando de forma geral, as bibliotecas PyInstaller, Selenium e CustomTkinter. Ela é capaz de registrar o estado dos campos selecionados para melhorar a experiencia do usuário.
